# Mythoi 🏛️

**Mythoi** is a growing digital library dedicated to Greek mythology — designed to make ancient stories accessible, engaging, and meaningful for modern readers.

🌐 Website: https://mythoi.org

---

## 📖 About the Project

Greek mythology is more than just stories about gods and heroes.

Mythoi focuses on:
- Clear, engaging storytelling
- Mythological symbolism and deeper meanings
- Connections between ancient myths and modern life

---

## ✨ Features

- Detailed articles on gods, heroes, and myths  
- SEO-optimized content structure  
- Clean, modern design  
- Educational and beginner-friendly  

---

## 🚀 Goal

To build the most accessible and engaging online resource for Greek mythology.
